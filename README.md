## Key Point

`branch`

```bash
# branch 확인
git branch

# branch 생성
git branch <name>

# branch 삭제
git branch -D <name>

# branch 이동
git switch <name>

# branch 생성 + 이동
git switch -c <name>
```

`push`

```bash
git push --set-upstream origin <name>
```

`merge`

```bash
git merge <name>
:q
git merge <name> -m <message>
```
